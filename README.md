# Amazon Recommendation System

## Overview
This project implements a recommendation system using Amazon product ratings data to suggest relevant items based on user preferences. The focus is on building and evaluating collaborative filtering–based recommendations using real-world data.

---

## Problem Statement
E-commerce platforms rely heavily on recommendation systems to improve user engagement and sales. This project explores how user–item interaction data can be leveraged to generate meaningful product recommendations at scale.

---

## Dataset
- **Source**: Amazon Reviews Dataset (Electronics category)
- **Provider**: UC San Diego (Julian McAuley)

Dataset reference:  
https://nijianmo.github.io/amazon/index.html

---

## Approach
- Data loading and preprocessing
- Exploratory data analysis (EDA)
- User–item interaction matrix construction
- Collaborative filtering–based recommendation logic
- Evaluation of recommendation outputs

---
